# DalamudPlugins

个人维护并开发的一些 Dalamud 插件。

这个仓库用于维护自定义 Dalamud 插件源，方便通过卫月的额外仓库功能安装与更新这里发布的插件。

## 仓库地址

```text
https://raw.githubusercontent.com/endfish/DalamudPlugins/main/repo.json
```

## 使用说明

1. 在游戏内输入 `/xlsettings`
2. 打开 `Experimental`
3. 将上面的仓库地址添加到额外仓库列表
4. 打开插件安装器后，即可查看并安装这里维护的插件
